Professional Bibliographic Standards

IFLA develops and maintains international standards that enable libraries worldwide to:

  • Create consistent and interoperable bibliographic records
  • Share resources effectively across institutions
  • Implement modern cataloging practices
  • Support discovery and access to library materials

IFLA Standards

Explore our comprehensive collection of international bibliographic standards, developed through collaborative efforts with library professionals worldwide.

ISBD for Manifestation (ISBDM)

Published

International Standard Bibliographic Description for Manifestation provides rules for creating consistent bibliographic descriptions of library materials in their physical or digital form.

View ISBDM Standard

Library Reference Model (LRM)

Published

A high-level conceptual model that provides a framework for understanding the bibliographic universe and the relationships between bibliographic entities.

View LRM Standard

International Standard Bibliographic Description (ISBD)

Development

The foundational standard for bibliographic description, providing rules for creating consistent and comprehensive bibliographic records across all types of library materials.

View ISBD Standard

Functional Requirements (FR)

Development

Specifications for functional requirements that support discovery, identification, selection, and access to bibliographic resources.

View FR Standard

Multilingual Dictionary of Cataloguing Terms (MulDiCat)

Development

A comprehensive multilingual dictionary providing standardized cataloguing terminology to support international library cooperation.

View MulDiCat Standard

UNIMARC

Development

Universal MARC format designed to facilitate the international exchange of bibliographic data and support library automation.

View UNIMARC Standard
