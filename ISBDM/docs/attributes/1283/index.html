has regional encoding

Element Reference​

  • Attribute:Value
  • JSON-LD
  • Turtle
  • RDF/XML
Definition
Relates a manifestation to a physical characteristic that is a designation that indicates that content that is embodied in the manifestation is encoded to restrict access to one or more regions of the world.
Scope note
This element applies to a manifestation that is a videodisc.
Domain
Manifestation

Additional information​

Stipulations​

Use the element super-type to record a less specific attribute.

Apply the stipulations for the element super-type unless indicated otherwise.

Record a value that appears in the ISBDM Regional Encoding value vocabulary, in the language or script of choice of the cataloguing agency.
