Sensory Specification Production Vocabulary

This example demonstrates the VocabularyTable component using the sensory-specification.csv file, which contains the production vocabulary with extensive multilingual support across 9 languages.

Features Demonstrated​

  • Production CSV Data: Real vocabulary data used in production systems
  • Extensive Multilingual Support: Available in English, French, Spanish, Italian, Russian, Bulgarian, Chinese, Croatian, and Latvian
  • Alternative Labels: Multiple alternative labels in different languages
  • Language Detection: Automatic detection of available languages from CSV data
  • Robust Internationalization: Demonstrates real-world multilingual vocabulary management

Technical Details​

CSV File: /static/data/CSV/sensory-specification.csv

Key Features:

  • Real-world production vocabulary data
  • Extensive multilingual coverage (9 languages)
  • Alternative labels for improved findability
  • Standardized SKOS structure
  • Used in international library systems

CSV Structure:

  • Complex multilingual column structure with language-specific arrays
  • Alternative labels with indexed notation (skos:altLabel@fr[0], skos:altLabel@fr[1])
  • Comprehensive definition coverage across all major languages
  • Proper SKOS Concept typing and hierarchical relationships

This example shows how the VocabularyTable component handles complex, real-world multilingual vocabulary data used in production library systems.
