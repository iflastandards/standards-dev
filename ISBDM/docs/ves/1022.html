ISBDM Category of Carrier value vocabulary

For use with element: has category of carrier

This value vocabulary is a source of values for a has category of carrier element.

The vocabulary covers the full scope of the element and will not be extended with additional values.

Value
Definition
Scope note

aperture card
(isbdm:1022#t1000)

A carrier type that consists of a card with one or more rectangular openings or apertures holding frames of microfilm.

audio belt
(isbdm:1022#t1001)

A carrier type that consists of a loop of flexible plastic or magnetic film on which audio signals are mechanically recorded, commonly known under the trade name Dictabelt.

audio cartridge
(isbdm:1022#t1002)

A carrier type that consists of a cartridge containing an audio tape.

audio cylinder
(isbdm:1022#t1003)

A carrier type that consists of a roller-shaped object on which sound waves are incised or indented in a continuous circular groove.

Wax cylinders, wire cylinders, etc., are included.

audio disc
(isbdm:1022#t1004)

A carrier type that consists of a disc on which sound waves, recorded as modulations, pulses, etc., are incised or indented in a continuous spiral groove.

audio roll
(isbdm:1022#t1005)

A carrier type that consists of a roll of paper on which musical notes are represented by perforations, designed to mechanically reproduce the music when used in a player piano, player organ, etc.

audio wire reel
(isbdm:1022#t1006)

A carrier type that consists of a reel or spool of steel or stainless steel wire upon which audio signals are magnetically recorded.

audiocassette
(isbdm:1022#t1007)

A carrier type that consists of a cassette containing an audiotape.

audiotape reel
(isbdm:1022#t1008)

A carrier type that consists of an open reel holding a length of audiotape to be used with reel-to-reel audio equipment.

card
(isbdm:1022#t1009)

A carrier type that consists of a small sheet of opaque material.

computer card
(isbdm:1022#t1010)

A carrier type that consists of a card containing digitally encoded data designed for use with a computer.

computer chip cartridge
(isbdm:1022#t1011)

A carrier type that consists of a cartridge containing a miniaturized electronic circuit on a small wafer of semiconductor silicon, designed to provide additional processing, memory, or storage capacity.

computer disc
(isbdm:1022#t1012)

A carrier type that consists of a disc containing digitally encoded data that are magnetically or optically recorded.

computer disc cartridge
(isbdm:1022#t1013)

A carrier type that consists of a cartridge containing one or more computer discs.

computer tape cartridge
(isbdm:1022#t1014)

A carrier type that consists of a cartridge containing a computer tape.

computer tape cassette
(isbdm:1022#t1015)

A carrier type that consists of a cassette containing a computer tape.

computer tape reel
(isbdm:1022#t1016)

A carrier type that consists of an open reel holding a length of computer tape to be used with a computer tape drive.

film cartridge
(isbdm:1022#t1017)

A carrier type that consists of a cartridge containing a motion picture film.

film cassette
(isbdm:1022#t1018)

A carrier type that consists of a cassette containing a motion picture film.

film reel
(isbdm:1022#t1019)

A carrier type that consists of an open reel holding a motion picture film to be used with a motion picture film projector.

film roll
(isbdm:1022#t1020)

A carrier type that consists of a wound length of film.

filmslip
(isbdm:1022#t1021)

A carrier type that consists of a short strip of film, usually in rigid format rather than rolled.

filmstrip
(isbdm:1022#t1022)

A carrier type that consists of a roll of film, with or without recorded sound, containing a succession of images intended for projection one at a time.

filmstrip cartridge
(isbdm:1022#t1023)

A carrier type that consists of a cartridge containing a filmstrip.

flipchart
(isbdm:1022#t1024)

A carrier type that consists of a hinging device holding two or more sheets designed for use on an easel.

microfiche
(isbdm:1022#t1025)

A carrier type that consists of a sheet of film bearing a number of micro-images in a two-dimensional array.

microfiche cassette
(isbdm:1022#t1026)

A carrier type that consists of a cassette containing uncut microfiches.

microfilm cartridge
(isbdm:1022#t1027)

A carrier type that consists of a cartridge containing a microfilm.

microfilm cassette
(isbdm:1022#t1028)

A carrier type that consists of a cassette containing a microfilm.

microfilm reel
(isbdm:1022#t1029)

A carrier type that consists of an open reel holding a microfilm, to be threaded into a microfilm reader.

microfilm roll
(isbdm:1022#t1030)

A carrier type that consists of a wound length of microfilm.

microfilm slip
(isbdm:1022#t1031)

A carrier type that consists of a short strip of microfilm cut from a roll.

microopaque
(isbdm:1022#t1032)

A carrier type that consists of a card or sheet of opaque material bearing a number of micro-images in a two-dimensional array.

microscope slide
(isbdm:1022#t1033)

A carrier type that consists of a small sheet of transparent material, with or without a protective mount, bearing a minute object designed for use with a device such as a microscope.

object
(isbdm:1022#t1034)

A carrier type that consists of a three-dimensional artefact, a replica of an artefact, or a naturally-occurring object.

online resource
(isbdm:1022#t1035)

A carrier type that consists of a digital resource accessed by means of hardware and software connections to a communications network.

overhead transparency
(isbdm:1022#t1036)

A carrier type that consists of a sheet of transparent material, with or without a protective mount, bearing an image designed for use with an overhead projector.

roll
(isbdm:1022#t1037)

A carrier type that consists of a wound length of paper, film, tape, etc.

sheet
(isbdm:1022#t1038)

A carrier type that consists of a flat, thin piece of paper, plastic, etc.

slide
(isbdm:1022#t1039)

A carrier type that consists of a small sheet of transparent material, usually in a protective mount, bearing an image designed for use with a slide projector or viewer.

sound-track reel
(isbdm:1022#t1040)

A carrier type that consists of an open reel holding a length of film on which sound is recorded.

Use for sound-track films, whether or not they are intended to accompany visual images on film.

stereograph card
(isbdm:1022#t1041)

A carrier type that consists of a card bearing stereographic images.

stereograph disc
(isbdm:1022#t1042)

A carrier type that consists of a disc with openings around the perimeter holding pairs of still images designed for use with a stereograph viewer.

video cartridge
(isbdm:1022#t1043)

A carrier type that consists of a cartridge containing a video tape.

videocassette
(isbdm:1022#t1044)

A carrier type that consists of a cassette containing a video tape.

videodisc
(isbdm:1022#t1045)

A carrier type that consists of a disc on which video signals, with or without sound, are recorded.

videotape reel
(isbdm:1022#t1046)

A carrier type that consists of an open reel holding a video tape for use with reel-to-reel video equipment.

volume
(isbdm:1022#t1047)

A carrier type that consists of one or more sheets bound or fastened together to form a single unit.
