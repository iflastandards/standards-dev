Glossary

The draft glossary contains terms that are used in stipulations and guidance.

The full glossary will also include entity and element labels and value vocabulary terms, with definitions and scope notes.

Value
Definition
Scope note

access point

An appellation of an instance of an entity in natural language that is selected from an authority file or constructed using a string encoding scheme by a cataloguing agency.

acronym

An initialism that is spoken as a word.

An acronym is not spoken as individual letters.

aggregate

A manifestation that embodies more than one discrete expression.

aggregating expression

The one and only expression that realises an aggregating work.

aggregating work

A work that is the intention or plan to select and arrange the expressions that are embodied in an aggregate.

An aggregating work is realised by one and only one expression that is also embodied in the aggregate.

alternative title

A distinct title that is appended to another title in a statement of title and responsibility.

An alternative title is separated from the other title by the word "or" or its equivalent, or by parentheses or similar punctuation.

appellation

A string that is used as a label to distinguish and reference an instance of an entity.

augmented expression

An expression of a single independent work that is embodied in an augmented expression aggregate and supplemented by expressions of one or more dependent works.

augmented expression aggregate

An aggregate that embodies an expression of a single independent work that is supplemented by augmenting expressions of one or more dependent works.

The augmented expression is supplemented with additional content that is not integral to the original work and does not significantly change the expression.

augmented work

A work that is realised by an augmented expression.

augmenting expression

An expression of a dependent work that is embodied in an augmented expression aggregate to supplement an augmented expression.

augmenting work

A work that is realised by an augmenting expression.

authorized access point

An access point that is selected for preference within an authority file.

authority file

A dataset that is a controlled set of values for identifying an entity.

An authority file usually includes access points and identifiers, as well as other metadata for each entity.

base access point

An access point that has no qualifiers for disambiguation or secondary collation.

chronogram

A word or phrase in which specific letters are intended to be interpreted as numerals that stand for a date when rearranged.

A chronogram is usually presented in a script where letters are assigned a numerical value.

collected expression

An expression of an independent work that is embodied in a collected expressions aggregate with expressions of other independent works.

collected expressions aggregate

An aggregate that embodies a set of expressions of more than one independent work.

The expressions may be selected according to a set of common characteristics that apply to the expressions or their works.

collecting expression

The one and only expression that realises a collecting work.

collecting work

A work that is the intention or plan to gather together two or more items for a purpose.

A purpose includes easy and contextual access to manifestations with characteristics in common, circulation or preservation management, and education, promotion, or entertainment.

collection manifestation

A manifestation that embodies a collecting work by holding the selected items.

collective title

A title that is the overall title of a manifestation that is an aggregate.

common title

A title that is the overall title of a manifestation that has sub-units or iterations.

compound title

A title that is composed of more than one other title or designation.

A compound title includes a combination of a title with an alternate title or a combination of a common title with a dependent title or dependent title designation.

conceptual part

A part of a work or the corresponding part of an expression that realises the work.

controlled string

A string that is created by a cataloguing agency to support information retrieval.

dataset

An aggregation of metadata that describes instances of specific entities.

Each instance has a discrete description. A dataset may include instances of more than one type of entity.

dereference

To obtain values of one or more characteristics of an instance of an entity that are associated with its IRI.

diachronic work

A work that is planned to be embodied through more than one act of publication or production over time.

element

A characteristic that typifies an entity.

A characteristic includes a general aspect such as a note, and a specific aspect such as an attribute or relationship.

element sub-type

A narrower category of an element.

element super-type

A broader category of an element.

element value

A string or IRI that is valid content of an element.

A value must conform to the definition and scope note of the element.

entity

An abstract class of a physical or conceptual thing in the universe of human discourse.

facsimile

Content that is reproduced in a new manifestation that preserves the layout and presentation of an existing manifestation.

A facsimile may or may not reproduce manifestation statements associated with the existing manifestation.

filename

A string that is used to label a computer file.

A filename includes a file extension that identifies the format of the file.

generic title

A general word or phrase for the kind, form, or genre of the content that is embodied in a manifestation.

identifier

A controlled string that is used to differentiate an instance of an entity within a local system.

An identifier string is intended for processing by machine and is created and assigned by an automated system. The scope of the system may be international.

information resource

A discrete source of recorded information.

initialism

A word that is based on the first letters of words in a phrase.

The word may be spoken as a word or as individual letters.

instance

A single discrete or individual example of an entity.

integrating diachronic work

A diachronic work that is embodied in the manifestation of a single expression.

The content of the manifestation is changed by the expression of an iteration of the diachronic work.

IRI

An internationalised resource identifier that is a machine-readable sequence of signs and symbols that is used to differentiate an entity, an instance of an entity, an element, or a concept within the global domain of the semantic web and open linked data.

An IRI is not processed as a string in an information retrieval system.

leaf

A single sheet that is bound or fastened with other sheets to form a volume.

nomen string

A string that is the appellation of an entity.

An appellation includes a name, title, access point, or identifier.

parallel expressions aggregate

An aggregate that embodies expressions of a single independent work.

Expressions in different languages and scripts are included.

parallel manifestation statement

A statement that appears in a manifestation in more than one language or script.

Parallel statements may be grouped by language or script, or by kind of statement.

produced manifestation

A manifestation that is exemplified by one and only one item that is created by using an artisanal or hand-made process.

A hand-made process includes handwriting, drawing, or typing.

published manifestation

A manifestation that is exemplified by one or more items that are manufactured by using a mechanical process.

A mechanical process includes a printing or digital manufacturing process.

qualitative value

A string that is a controlled term that appears in a value vocabulary.

quantitative value

A string that is a measurement that consists of a number and a qualitative value for the measurement unit.

A qualitative value for the measurement unit includes a word, phrase, abbreviation, or symbol.

resource entity

A work, expression, manifestation or item.

shortcut

An element that represents a chain of two or more elements and intermediary entities.

The intermediary entities are not recorded.

state

An intended variation in the manufacture of a manifestation that results in one or more exemplars that are distinct from other exemplars.

The variation includes a change to a printing plate.

static work

A work that is planned to be embodied through a single act of publication or production.

string

A sequence of signs or symbols that is treated as a whole.

Signs and symbols include alphanumeric and syllabic characters, ideographs, and punctuation marks.

string encoding scheme

A set of rules for creating a single string from a set of other strings.

Rules include the choice and sequence of source strings, punctuation between source strings, and the incorporation of controlled text.

stringify

A process that converts an IRI to a string identifier.

A process includes enclosing the IRI in a string delimiter such as quotation marks.

successive diachronic work

A diachronic work that is embodied in the manifestation by sequential sub-units.

The content of the manifestation is changed by adding expressions in sequential issues of the diachronic work.

title

An appellation of an instance of a resource entity in natural language and phrasing that is assigned by a creator of the instance.

type of composition

A form, genre, or generic term used frequently by different composers to refer to a kind of musical work.

uncontrolled string

A string that is not created by a cataloguing agency.

unit

A logical part of an intangible manifestation or a physical part of a tangible manifestation

value vocabulary

A set of controlled terms that are used to describe a specific characteristic of an entity.

whitespace

The area of a page that does not embody content.

Whitespace includes margins, indentations, and blank lines that appear in the layout of content.
